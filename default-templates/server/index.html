Welcome to Solid prototype

This is a prototype implementation of a Solid server. It is a fully functional server, but there are no security or stability guarantees. If you have not already done so, please register. When registering you will get a Pod and a WebID from solidcommunity.net. If you want to register on a different server check out the available list of providers.

  • Read more about Solid at https://solidproject.org/.
  • There is a vibrant community around Solid gathering on the forum, chat or at a Solid event.
  • If you are interested to host your own server, read about the Community Server.
